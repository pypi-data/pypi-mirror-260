# 坐享其成

## 下载

### Docker

```
docker pull apachecn0/zuoxiangqicheng
docker run -tid -p <port>:80 apachecn0/zuoxiangqicheng
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install zuoxiangqicheng
zuoxiangqicheng <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g zuoxiangqicheng
zuoxiangqicheng <port>
# 访问 http://localhost:{port} 查看文档
```
