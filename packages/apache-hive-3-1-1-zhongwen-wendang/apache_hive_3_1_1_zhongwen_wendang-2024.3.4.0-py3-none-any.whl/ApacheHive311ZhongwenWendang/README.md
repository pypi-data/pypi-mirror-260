# Apache Hive 3.1.1 中文文档

## 下载

### Docker

```
docker pull apachecn0/apache-hive-3-1-1-zhongwen-wendang
docker run -tid -p <port>:80 apachecn0/apache-hive-3-1-1-zhongwen-wendang
# 访问 http://localhost:{port} 查看文档
```

