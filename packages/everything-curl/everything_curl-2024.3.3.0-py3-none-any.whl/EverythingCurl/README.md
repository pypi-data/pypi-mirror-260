# Everything curl

## 下载

### Docker

```
docker pull apachecn0/everything-curl
docker run -tid -p <port>:80 apachecn0/everything-curl
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install everything-curl
everything-curl <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g everything-curl
everything-curl <port>
# 访问 http://localhost:{port} 查看文档
```
