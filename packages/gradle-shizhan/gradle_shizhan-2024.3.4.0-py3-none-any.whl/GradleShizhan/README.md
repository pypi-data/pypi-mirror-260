# Gradle 实战

## 下载

### Docker

```
docker pull apachecn0/gradle-shizhan
docker run -tid -p <port>:80 apachecn0/gradle-shizhan
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install gradle-shizhan
gradle-shizhan <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g gradle-shizhan
gradle-shizhan <port>
# 访问 http://localhost:{port} 查看文档
```
