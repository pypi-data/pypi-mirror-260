# Rxjs 5 ultimate

## 下载

### Docker

```
docker pull apachecn0/rxjs-5-ultimate
docker run -tid -p <port>:80 apachecn0/rxjs-5-ultimate
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install rxjs-5-ultimate
rxjs-5-ultimate <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g rxjs-5-ultimate
rxjs-5-ultimate <port>
# 访问 http://localhost:{port} 查看文档
```
