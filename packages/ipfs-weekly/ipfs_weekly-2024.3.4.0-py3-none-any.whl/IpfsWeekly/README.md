# IPFS Weekly

## 下载

### Docker

```
docker pull apachecn0/ipfs-weekly
docker run -tid -p <port>:80 apachecn0/ipfs-weekly
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install ipfs-weekly
ipfs-weekly <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g ipfs-weekly
ipfs-weekly <port>
# 访问 http://localhost:{port} 查看文档
```
