# Kivy中文编程指南

## 下载

### Docker

```
docker pull apachecn0/kivy-zhongwen-biancheng-zhinan
docker run -tid -p <port>:80 apachecn0/kivy-zhongwen-biancheng-zhinan
# 访问 http://localhost:{port} 查看文档
```

```
pip install kivy-zhongwen-biancheng-zhinan

# 访问 http://localhost:{port} 查看文档
```
