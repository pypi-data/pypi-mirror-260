# Flutter 实战

## 下载

### Docker

```
docker pull apachecn0/flutter-shizhan
docker run -tid -p <port>:80 apachecn0/flutter-shizhan
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install flutter-shizhan
flutter-shizhan <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g flutter-shizhan
flutter-shizhan <port>
# 访问 http://localhost:{port} 查看文档
```
