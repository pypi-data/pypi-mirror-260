# 简明 Python 教程 4.0

## 下载

### Docker

```
docker pull apachecn0/jianming-python-jiaocheng-4-0
docker run -tid -p <port>:80 apachecn0/jianming-python-jiaocheng-4-0
# 访问 http://localhost:{port} 查看文档
```

```

```
