# Advanced R

## 下载

### Docker

```
docker pull apachecn0/advanced-r
docker run -tid -p <port>:80 apachecn0/advanced-r
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install advanced-r
advanced-r <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g advanced-r
advanced-r <port>
# 访问 http://localhost:{port} 查看文档
```
